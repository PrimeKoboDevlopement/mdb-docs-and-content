Steppers Pro设定

下载并解压缩MDB程序包后，打开index.html文件。

然后，在链接的CSS文件下面添加Stepper文件。


<!-MDBootstrap Steppers Pro-> <link href="css/addons-pro/stepper.min.css" rel="stylesheet">

接下来，在链接的JavaScript文件下方添加Steppers文件。


<!-MDBootstrap Steppers Pro-> <script type="text/javascript" src="js/addons-pro/stepper.min.js"></script>

注意： 不要忘记初始化Steppers Pro。

$（document）.ready（function（）{$（&#39;。stepper&#39;）。mdbStepper（）;}）

表格可编辑

Vue Bootstrap 4表可编辑集成

使用可编辑的引导表，您可以添加和删除行以及更改单元格内的文本和信息。现在，基于JavaScript的网站上就地编辑变得更加轻松快捷。

要开始使用可编辑表格，请参见 “入门” 此页面上的标签。

注意： 表可编辑的集成从版本开始可用 4.5.0 （发布于2018年7月30日）。

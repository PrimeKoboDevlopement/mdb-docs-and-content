表格滚动-水平和垂直

Vue Bootstrap表滚动

对于具有大量数据的表，您可以使用滚动功能，作为分页的替代方法。

滚动功能可垂直（y轴）和水平（x轴）使用。

根据您使用的表类型（数据表或简单静态表），有两种不同的解决方案。

要开始使用我们的表，请参阅 “入门” 此页面上的标签。

注意： 此集成版本提供 4.5.0 （发布于2018年7月30日）。

入门：下载和设置

下载

所有组件和功能都是MDBootstrap for Vue软件包的一部分。

MDBootstrap（用于Bootstrap的材料设计）是一个免费的（由MIT许可）框架，该框架将Material Design和最新的Bootstrap 4与Vue结合在一起。

单击下面的按钮转到下载页面，您可以在此处下载MDBootstrap软件包。

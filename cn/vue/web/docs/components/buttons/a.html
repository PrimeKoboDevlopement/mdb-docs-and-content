按钮-API

在本节中，您将找到有关“按钮”组件的高级信息。您将了解此组件中需要哪些模块，配置该组件的可能性以及在使用该组件时可以使用的事件和方法。

导入账单

从“ mdbvue”导入{mdbBtn，mdbBtnFixed，mdbBtnFixedItem，mdbBtnGroup，mdbIcon}；

API参考：属性

名称 类型 默认 描述 范例
标签 字符串 按钮 更改按钮标签 <mdb-btn tag="a">
颜色 字符串 默认 确定按钮颜色 <mdb-btn color="primary">
大綱 字符串 更改按钮轮廓颜色 <mdb-btn outline="primary">
文本 字符串 关于按钮文本颜色等，可以使按钮自定义更加容易。只需添加我们的颜色调色板省略 -文本后缀-将为您添加后缀-一切就绪 <mdb-btn color="primary" text="secondary">
大小 字符串 确定按钮大小 <mdb-btn size="lg">
块 (block) 真假值 false 创建块级按钮 <mdb-btn块>
角色 字符串 将角色属性添加到按钮 <mdb-btn role =“ ...”>
类型 字符串 将类型属性添加到按钮 <mdb-btn type =“ ...”>
活动 真假值 false 激活的类 <mdb-btn有效>
禁用 真假值 false 添加禁用的类 <已禁用mdb-btn>
图标 字符串 在按钮内添加图标 <mdb-btn icon="user">
iconLeft 真假值 false 确定图标的位置 <mdb-btn icon左>
iconRight 真假值 false 确定图标的位置 <mdb-btn iconRight>
渐变 字符串 向按钮添加渐变效果 <mdb-btn gradient="peach">
圆形的 真假值 false 在按钮上添加圆角效果 <mdb-btn取整>
浮动 真假值 false 为按钮添加浮动效果 <mdb-btn浮动>
平面 真假值 false 为按钮添加平面效果 <mdb-btn flat>
透明 真假值 false 使按钮透明 <mdb-btn透明>
保存 真假值 false 选择内的保存按钮非常有用 <mdb-btn保存>
iconClass 字符串 在按钮内更改图标的元素类 <mdb-btn iconClass =“ ...” >
iconColor 字符串 更改图标的元素颜色 <mdb-btn iconColor =“ ...” >

Vue引导面板

Vue面板-Bootstrap 4和材料设计

Vue Bootstrap的面板提供了具有多个变体的灵活且可扩展的内容容器。这些选项包括但绝不限于页眉和页脚，各种内容，上下文背景颜色以及强大的显示选项。面板类似于卡，但不包括媒体。

基本示例

面板以尽可能少的标记和样式构建，但仍设法提供大量控制和自定义。它们与flexbox一起构建，可以轻松对齐并与其他Bootstrap组件很好地混合。他们没有余量默认情况下，所以使用间隔实用程序如所须。

以下是具有混合内容和固定宽度的基本面板的示例。面板没有固定的宽度开始，因此它们自然会填充其父元素的整个宽度。这很容易通过我们的各种定制尺寸选择 。

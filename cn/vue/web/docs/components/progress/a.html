API

进步

在本节中，您将找到有关“进度”组件的高级信息。您将了解此组件中需要哪些模块，配置该组件的可能性以及在使用该组件时可以使用的事件和方法。

导入账单

API参考：属性

名称 类型 默认 描述 范例
标签 字符串 div 更改进度标签 <mdb-progress tag="p" />
价值 号码 0 变更进度值 <mdb-progress :value="25" />
高 号码 更改进度高度 <mdb-progress :height="25" />
颜色 字符串 更改进度颜色 <mdb-progress color="warning" />
条纹卷 真假值 false 条纹效果 <mdb-progress带状/>
动画 真假值 false 动画效果 <mdb-progress动画/>
不确定 真假值 false 增加无限负载 <mdb-progress不确定/>

预加载脚本API

在本节中，您将找到有关“预加载脚本”组件的高级信息。

导入账单

API参考：属性

名称 类型 默认 描述 范例
颜色 字符串 白色 更改背景颜色 <mdb-preloading-script color="default" />
透明度 号码 1 更改背景的不透明度 <mdb-preloading-script :opacity="0.8" />
格局很大 真假值 false 将微调框的大小更改为大 <mdb-preloading-script big />
小 真假值 将微调框的尺寸更改为较小 <mdb-preloading-script small />
疯狂的 真假值 false 将微调器的动画速度更改为快速 <mdb-preloading-script crazy />
spinnerColor 字符串 蓝色 更改微调器的颜色 <mdb-preloading-script color="red" />
多色 真假值 false 将微调框的颜色主题更改为多色 <mdb-preloading-script multicolor />
载入中 真假值 如果为 true，则幻灯片将没有固定位置 只要可以看到预加载脚本装货属性设置为真正 <mdb-preloading-script :loading="isLoading" />
动画 字符串 淡出 更改预加载脚本的退出动画。所有动画的列表均可用这里 <mdb-preloading-script animation="fadeOutLeft" />

图标-API

在本节中，您将找到有关Icon组件的高级信息。您将了解此组件中需要哪些模块，配置该组件的可能性以及在使用该组件时可以使用的事件和方法。

导入账单

从“ mdbvue”导入{mdbIcon}；

API参考：图标属性

下表显示了Icon组件的配置选项。

名称 类型 默认 描述 范例
图标 字符串 必须! 您要使用的“真棒字体”图标名称图标列表 <mdb-icon icon="chevron" />
較遠的 / 较远的 (jiào yuǎn de) 真假值 false 设置常规图标类别； 别名：常规 <mdb-icon far icon =“ user” />
很棒 真假值 false 设置品牌图标类别； 别名：品牌 <mdb-icon fab icon =“ twitter” />
Fal 真假值 false 设置灯光图标类别； 别名：浅色 <mdb-icon fal icon =“ user” />
時尚潮流 真假值 false 设置双色调图标类别； 别名：浅色 <mdb-icon fad icon =“ user” />
brand 真假值 false 设置品牌图标类别； 别名：fab <mdb-icon brand icon =“ twitter” />
定期 真假值 false 设置常规图标类别； 别名：远 <mdb-icon常规图标=“ twitter” />
明亮 真假值 false 设置灯光图标类别； 别名：fal <mdb-icon light icon =“ twitter” />
双色调 真假值 false 设置双色调图标类别； 别名：时尚 <mdb-icon duotone icon =“ twitter” />
大小 [布尔值，字符串] false 设置图标大小 <mdb-icon size="2x" />
固定 真假值 false 设置图标固定选项 <mdb-icon已修复/>
拉取 [布尔值，字符串] false 设置图标拉选项 <mdb-icon pull />
边界 真假值 false 图标边框 <mdb-icon边框/>

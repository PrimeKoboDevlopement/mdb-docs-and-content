入门：下载和设置

下载

所有组件和功能都是MDBootstrap for Vue软件包的一部分。

MDBootstrap（用于Bootstrap的材料设计）是一个免费的（由MIT许可）框架，该框架将Material Design和最新的Bootstrap 4与Vue结合在一起。

单击下面的按钮转到下载页面，您可以在此处下载MDBootstrap软件包。

API参考：属性

名称 类型 默认 描述 范例
交换柱 阵列 - CSets表的列 <mdb-editable :columns="columns"/>
行 阵列 设置表格的行 <mdb-editable :rows="rows" />
带边框 真假值 false 在表格内容周围显示边框 <mdb可编辑的带边框的/>
无边框 真假值 false 删除边框样式 <mdb可编辑的无边界/>
黑暗 真假值 false 将表格样式更改为深色背景 <mdb可编辑的深色/>
分页 真假值 false 如果设置为真正分页 <mdb可编辑的分页/>
没有精选条目符合标准。 号码 10 确定页面上显示的最大条目数 <mdb可编辑分页：entries =“ 20” />
显示屏 号码 5 设置分页中的页面项数 <mdb可编辑的分页：display =“ 4” />
相应迅速 真假值 false 启用响应行为 <mdb-editable响应/>
响应式 真假值 false 将敏感的断点设置为小型设备 <mdb-editable响应Sm />
响应式 真假值 false 将响应断点设置为中型设备 <mdb可编辑的ResponseMd />
反应性 真假值 false 将响应断点设置为大型设备 <mdb-editable响应Lg />
响应式 真假值 false 将响应断点设置为超大型设备 <mdb-editable响应Xl />

Vue引导显示

Vue展示-Bootstrap 4和材料设计

通过我们的显示实用程序，可以快速响应地切换组件的显示值，以及更多。包括对某些更常见值的支持，以及一些在打印时控制显示的附加功能。

工作原理

更改值显示属性与我们的响应显示实用程序类。我们特意仅支持以下所有可能值的子集show 。您可以根据需要组合类以实现各种效果。

注释

显示适用于所有对象的实用程序类断点 ，来自 xs 至 l ，其中没有断点的缩写。这是因为这些类是从最小宽度：0； 以及更高版本，因此不受媒体查询的约束。但是，其余的断点确实包含断点的缩写。

因此，这些类使用以下格式命名：

  • .d- {value} 对于 xs
  • .d- {breakpoint}-{value} 对于短信 ， md ， lg 和 l 。

哪里值是其中之一：

  • 无
  • 内联
  • 内嵌块
  • 块 (block)
  • 表
  • 表单元格
  • 表行
  • 弯曲
  • 内联块

媒体查询会影响具有给定断点的屏幕宽度或更大 。例如， .d-lg-none 套显示：无； 双方 lg 和 l 屏幕。

隐藏元素

为了更快地进行移动友好型开发，请使用响应式显示类来按设备显示和隐藏元素。避免为同一网站创建完全不同的版本，而应针对每个屏幕尺寸相应地隐藏元素。

要隐藏元素，只需使用 .d-none 类或其中之一 .d- {sm，md，lg，xl}-无用于任何响应屏幕变化的类。

要仅在给定的屏幕尺寸间隔上显示元素，可以组合一个 .d-*-无一个班 .d-*-*类 ，例如 .d-none .d-md-block .d-xl-none 将隐藏所有屏幕尺寸的元素（中型和大型设备除外）。
